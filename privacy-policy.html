Memio – Privacy Policy

Last updated: July 2026

Memio is a browser extension that lets you save highlighted text and notes directly to Obsidian or Notion without leaving the page. It can also optionally generate a title for a note using an AI provider you configure. This policy explains what data Memio handles and how.

What data Memio handles

  • Text you highlight on a webpage
  • Notes you write within the extension
  • API tokens you configure (Notion API key, Obsidian Local REST API key, and — if enabled — an API key for an AI provider)

How your data is used

Highlighted text and notes are sent only to destinations you personally configure: your own Obsidian vault via the Local REST API, your own Notion workspace via your personal API token, and — only if you enable the optional AI title feature — the AI provider you choose (Anthropic Claude, OpenAI, or Google Gemini), using an API key you supply. No data is sent to any server operated by the developer.

Optional AI title generation

Memio can suggest a title for a note using an AI provider. This feature is off by default and only activates once you add your own API key for a provider (Claude, ChatGPT, or Gemini) under Settings. When used, the text of that note is sent directly from your browser to the AI provider you selected, under your own account and usage terms with that provider — not to any server operated by the developer.

API token storage

Your API tokens are stored locally using Chrome's built-in storage API. They are never transmitted to any external server beyond the integration or AI endpoint you have personally configured.

Analytics and tracking

Memio collects no analytics, usage data, or telemetry of any kind.

Third parties

Memio does not share any data with third parties beyond the destinations you configure yourself: your own Obsidian vault, your own Notion workspace, and — only if enabled — your chosen AI provider (Anthropic, OpenAI, or Google).

Contact
